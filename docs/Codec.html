Codec

Codec

Encode and decode data from ETH JSON RPC. This class is a wrapper of ethers.Interface.

Constructor

new Codec(abi)

Source:
  • codec.js, line 23

Initialize by Abi.create and Abi.fromJsonFile.

Example
let abi = require('abi.json')
let codec = new Codec(abi)

// Retrieve topic for filtering logs.
let topic = codec.getEventTopic(eventName)

// Retrieve data from a log.
let data = codec.parseLog(log)
