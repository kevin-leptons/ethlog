RequestValve

RequestValve

Control quota, log range for requests to Ethereum node. Also lock queries for critical errors such as bad request, bad response and overloading.

Constructor

new RequestValve(config)

Source:
  • safe_node.js, line 184

Initialize by RequestValve.create.

Example
let valve = RequestValve.create()

// Update state by.
valve.updateLogRange()
valve.downLogRange()
value.reportError()

// Ask for performing requests to Ethereum node.
valve.open()

// Retrieve estimated log range.
valve.logRange()
